El plugin RecordatorioCS permite enviar un nombre de evento en una fecha y hora determinada a na direccion de correo electronico.

== Description ==

El plugin RecordatorioCS  permite el registro de varios recordatorios en diferentes fechas y horas, los cuales seran enviados al correo electronico,
para ello se debe describir el nombre del recordatorio, seleccionar la fecha y hora que desea.
Tambien es necesario configurar un servidor de correo electronico para el envio correcto de emails.

== Installation ==
1. Se debe configurar el servidor de correo electronico para realizar el envio de correos
2. El Plugin se debe buscar en la pestania Plugin de Wordpress con la palabra RecordatorioCS
2. Descarga y Activa el PLUGIN
3. En el menu que se crea a la izquierda hacer click y se muestra la pagina correspondiente
